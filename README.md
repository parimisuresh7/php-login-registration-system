# PHP User Login & Registration System

A simple authentication system built with PHP and MySQL.

## Features
- User Registration
- Secure Login
- Password Hashing
- Session Authentication
- Logout System

## Technologies
- PHP
- MySQL
- HTML
- CSS

## Installation

1. Clone repository
2. Import database.sql
3. Update database credentials in config/database.php
4. Run project on localhost
